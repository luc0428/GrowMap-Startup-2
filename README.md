# GrowMap - Analise e desenvolvimentos de Habilidades

O GrowMap é uma **plataforma web** desenvolvida em **Angular, TypeScript, HTML e CSS** voltada para **gestão e análise de colaboradores**.  
Com ele, gestores podem acompanhar o desenvolvimento de habilidades, identificar pontos fortes e áreas que precisam de aprimoramento, e tomar decisões mais estratégicas sobre seus recursos humanos.

---

## Funcionalidades principais

- **Cadastro e gerenciamento de colaboradores:** Armazene informações detalhadas de cada membro da equipe.  
- **Avaliação de habilidades:** Registre competências técnicas e comportamentais de cada colaborador.  
- **Análise de desenvolvimento:** Relatórios e gráficos mostram evolução, lacunas de habilidades e potencial de crescimento.  
- **Suporte à tomada de decisão:** Ajuda gestores a planejar treinamentos, promoções e alocação de recursos de forma estratégica.

---

## Sobre o projeto

O GrowMap é desenvolvido por uma equipe de **três alunos**, como parte de um trabalho colaborativo.  
O desenvolvimento segue boas práticas de versionamento com **Git e GitHub**, utilizando **branches** para organizar tarefas e integrar funcionalidades de forma contínua.

Este projeto oferece uma oportunidade de aplicar conceitos de **desenvolvimento front-end moderno**, **análise de dados de RH** e **trabalho em equipe**, servindo tanto como aprendizado prático quanto como portfólio técnico.
